relativize()method
public function relativize(reference:URI):URI

Relativize a reference uri against this URI as the base URI according to RFC 3986. If both scheme and authority are equal, this method always returns a URI with a relative path, even it an absolute path would be shorter.

Parameters
reference:URI — the URI to relativize. Must be an absolute URI.

Returns
URI — an URI which yields the reference URI when resolved against the given base URI

Throws
URIError — if this base or the given reference is a relative URI
