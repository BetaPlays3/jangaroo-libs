relativize()method 
public static function relativize(base:String, reference:String):String

Relativize a reference uri against a base URI according to RFC 3986. If both scheme and authority are equal, this method always returns a URI with a relative path, even it an absolute path would be shorter.

Parameters
base:String — the base URI. Must be an absolute URI.
 
reference:String — the URI to relativize. Must be an absolute URI.

Returns
String — an URI which yields the reference URI when resolved against the given base URI

Throws
URIError — if base or reference is a relative URI
