Packageext.state
Classpublic class CookieProvider
InheritanceCookieProvider Provider

The default Provider implementation which saves state via cookies.
Usage: 

   var cp = new Ext.state.CookieProvider({
       path: "/cgi-bin/",
       expires: new Date(new Date().getTime()+(10000040)), //30 days
       domain: "extjs.com"
   });
   Ext.state.Manager.setProvider(cp);
