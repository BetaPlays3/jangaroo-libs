getBy()method 
public function getBy(fn:Function, scope:Object):Array

Returns zero or more windows in the group using the custom search function passed to this method. The function should accept a single ext.Window reference as its only argument and should return true if the window matches the search criteria, otherwise it should return false.

Parameters
fn:Function — The search function
 
scope:Object — The scope in which to execute the function (defaults to the window that gets passed to the function if not specified)

Returns
Array — An array of zero or more matching windows
