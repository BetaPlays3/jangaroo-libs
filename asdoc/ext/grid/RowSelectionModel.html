Packageext.grid
Classpublic class RowSelectionModel
InheritanceRowSelectionModel AbstractSelectionModel Observable
SubclassesCheckboxSelectionModel

The default SelectionModel used by Ext.grid.GridPanel. It supports multiple selections and keyboard selection/navigation. The objects stored as selections and returned by getSelected, and getSelections are the Records which provide the data for the selected rows.

public function selectRow(row:Number, keepExisting:Boolean, preventViewNotify:Boolean):void

Selects a row. Before selecting a row, checks if the selection model locked and fires the beforerowselect event. If these checks are satisfied the row will be selected and followed up by firing the rowselect and selectionchange events.

Parameters
