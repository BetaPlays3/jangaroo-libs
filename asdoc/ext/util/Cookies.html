get()method 
public static function get(name:Object):*

Retrieves cookies that are accessible by the current page. If a cookie does not exist, get() returns null. The following example retrieves the cookie called "valid" and stores the String value in the variable validStatus. 


     var validStatus = ext.util.Cookies.get("valid");

Parameters
name:Object — The name of the cookie to get

Returns
* — Returns the cookie value for the specified name; null if the cookie name does not exist.
