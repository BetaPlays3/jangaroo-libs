initKeyEvent()method
public function initKeyEvent(type:String, bubbles:Boolean, cancelable:Boolean, viewArg:Object, ctrlKeyArg:Boolean, altKeyArg:Boolean, shiftKeyArg:Boolean, metaKeyArg:Boolean, keyCodeArg:uint, charCodeArg:uint):void

Initialize the value of this KeyEvent.

Parameters
type:String — The type of event.
 
bubbles:Boolean — A boolean indicating whether the event should bubble up through the event chain or not (see bubbles).
 
cancelable:Boolean — A boolean indicating whether the event can be canceled (see cancelable).
 
viewArg:Object — Specifies UIEvent.view. This value may be null.
 
ctrlKeyArg:Boolean — bool True if the Virtual Key to be generated is a combination of the Ctrl key and other keys
 
altKeyArg:Boolean — bool True if the Virtual Key to be generated is a combination of the Alt key and other keys
 
shiftKeyArg:Boolean — bool True if the Virtual Key to be generated is a combination of the Shift key and other keys
 
metaKeyArg:Boolean — bool True if the Virtual Key to be generated is a combination of the Meta key and other keys
 
keyCodeArg:uint — unsigned long the virtual key code value of the key which was depressed, otherwise zero
 
charCodeArg:uint — unsigned long the Unicode character associated with the depressed key otherwise zero
